# Newa:SayQ - Nepal Bhasa Learning App

**Newa:SayQ** is a mobile app designed to help users learn **Nepal Bhasa** using modern techniques such as **Reinforcement Machine Learning** and **Spaced Repetition**. Built with **React Native**, the app will initially target **Android** devices, with plans to support iOS in the future

## Features

- Interactive lessons and exercises
- Personalized learning powered by ML
- Spaced repetition for effective memorization

## Getting Started

This project uses React Native. To get started, clone the repo and follow the setup instructions in the documentation.
